    AgentDesk 使用参数数组直接启动进程，不经过 shell，也不会自动下载注册表里的软件。具体参数以该 Agent 官方 ACP 文档为准。

    AgentDesk

    管理多个 Claude / Codex 账号，不保存任何密码 —— 登录始终发生在官方 App 里。

    1. 新增一个账号槽位（工作号 / 个人号 / 备用号……）。
    2. 选中它点打开账号，在弹出的官方 App 里登录。
    3. 用过之后，这个账号的本地会话会自动出现在中间列表。
    4. 选中会话点复制交接信息，粘给新会话继续上下文。

    每个槽位是独立登录、互不挤下线，可加备注和分组管理。读不到会话时点诊断，或用路径手动指定会话目录。
